# Restaurant Reviews React App + AWS deployment

This is a React App that uses Tailwind CSS for styling and AWS for deployment.

```
$ npm install
$ npm run frontend:dev
```

To debug the project locally

```
$ npm run dev
```

To deploy the build manually

```
$ npm run build
```

Preview app - deployed in AWS Cloudfront - https://d2kedliktzuk92.cloudfront.net/
